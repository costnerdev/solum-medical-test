# Solum Medical Test

Download this repository and run commands as described below

## Directory A

1. Navigate to project directory ```cd A```
2. Run the program ```python main.py``` or ```python3 main.py```
3. Enter the required input when prompted.

## Directory B

1. Navigate to project directory ```cd B```
2. Run the program ```python main.py``` or ```python3 main.py```
3. Enter the required input when prompted.

## Directory C

1. Navigate to project directory named *frontend* ```cd C/frontend```
2. Run the program ```npm run dev```
3. Play around with the simple login form  ```http://localhost:3000/sign-in``` and dashboard ```http://localhost:3000/```
